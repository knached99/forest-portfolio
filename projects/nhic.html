NHIC School Management System

A full-featured school management platform designed for NHIC to support digital education. The system enables teachers to deliver online learning, parents to monitor student progress, and administrators to manage school operations efficiently. Currently in beta development.

PHP / Laravel JavaScript HTML CSS MySQL

Key Features

  • Online learning platform for students
  • Parent dashboards for progress tracking
  • Teacher content and class management
  • Admin tools for school operations
